# OurCity CLI

This is a command-line interface for OurCity. This secondary frontend is proof of our decoupled architecture of the backend server.

## Setup

1. Install dependencies:

```bash
pip install -r requirements.txt
```

2. Make sure the backend server is running (default: `http://localhost:8000`)

## Usage

Run the CLI:
```bash
python ourcity_cli.py
```

Once ran, type 'help' for available commands or 'exit' to quit
